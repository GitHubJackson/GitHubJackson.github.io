发表于 2018-12-17 | 更新于 2019-02-28 | 分类于 Javascript | 阅读次数：

ECMAScript6 是 JavaScript 的标准，目前浏览器的 JavaScript 大多数都支持ES6。ES6使得 JavaScript 语言可以用来编写复杂的大型应用程序，成为企业级开发语言。

var,let和const

  • var 和 let 都可用于定义变量，但是 let 定义的变量只能在一个作用域内使用
  • const 定义不可变量

js 在引擎扫描代码时，如果发现变量声明，用 var 声明变量时会将声明提升到函数或全局作用域的顶部。但是 let 或者 const，会将声明关进一个小黑屋也是TDZ(暂时性死区)，只有执行到变量声明这句语句时，变量才会从小黑屋被放出来，才能安全使用这个变量
