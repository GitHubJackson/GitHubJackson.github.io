ctx(Context对象)，即上下文

Koa Context 将 node 的 request 和 response 对象封装到单个对象中，注意ctx.body即ctx.response.body
具体查看官网

中间件机制

Koa中使用app.use()用来加载中间件，基本上Koa 所有的功能都是通过中间件实现的。每个中间件默认接受两个参数，第一个参数是 Context 对象，第二个参数是next函数。只要调用next函数，就可以把执行权转交给下一个中间件
