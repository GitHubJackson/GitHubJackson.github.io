基于vux的底部固定导航栏和点击切换颜色

发表于 2019-01-03 | 分类于 vue | 阅读次数：

vux是基于vue的移动端ui框架，有很丰富的组件供使用。vux的Tabbar组件可用于实现底部导航菜单栏，下面将该组件升级了一下，使之能在页面滚动情况下固定导航栏和点击导航图标切换图标颜色。

搭建vux项目的几个基本步骤：

  • npm install vue-cli -g(如果还没安装脚手架，先安装)
  • vue init airyland/vux2 projectPath(自定义项目名称)
  • cd projectPath
  • npm install(建议使用cnpm install)
  • npm run dev
