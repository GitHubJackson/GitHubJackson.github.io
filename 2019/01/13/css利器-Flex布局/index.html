css利器--Flex布局

发表于 2019-01-13 | 分类于 css | 阅读次数：

之前兼职做项目时，用了挺多浮动(float)，结果有些没消除，对后面的布局产生了影响，我前辈就跟我说，以后浮动用flex去实现就行，不要用float了。flex布局的确是CSS的布局利器，尤其是在移动端网页设计上。

注意：在webkit内核的浏览器中使用时，必须加上*-webkit-*前缀

重点属性

父容器属性

  • flex-direction 主轴的方向，即项目排列的方向

row(默认)|row-reverse|column|column-reverse

  • flex-wrap 决定是否换行以及换行的方式
  • flex-flow 是flex-direction和flex-wrap的简写形式
  • justify-content 项目在主轴方向上排列的方式

  • align-items 项目在交叉轴上的对齐方式

  • align-content 有多根主轴时，即item不止一行时，多行在交叉轴轴上的对齐方式，此时align-items属性将失效

项目属性

  • flex

flex:1 1 30%

分配细则见深入理解Flex布局 – flex-grow & flex-shrink & flex-basis

  • order 项目排列顺序的优先级，数值越低，优先级越高
  • align-self 允许item有自己独特的在交叉轴上的对齐方式
