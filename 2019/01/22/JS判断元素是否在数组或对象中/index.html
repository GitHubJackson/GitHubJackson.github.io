JS判断元素是否在数组或对象中

发表于 2019-01-22 | 更新于 2019-01-23 | 分类于 Javascript | 阅读次数：

学js的时候没把这个放心上，实际做项目发现，后台给的数据多是对象数组，有时候要判断其中的对象是否存在，这点跟普通数组还是有差别的。

普通数组

  • 用for循环遍历数组，判断数组是否存在，最笨也是最基本的方法

  • indexOf(n) 返回数组中n的索引值，若不存在返回-1

  • forEach

对象数组

  • 使用数组some方法判断，检测数组中的元素是否有满足指定条件的，若满足返回true，否则返回false。只要有一个元素满足条件，就返回true，剩余的元素不会再遍历。
  • every 方法参数跟some一样，不同的是数组中检测到有一个元素不满足，就返回false，且剩余的元素不会再遍历。如果所有元素都满足条件，就返回true

    对象

  • hasOwnProperty方法返回一个布尔值，判断对象是否包含特定的自身（非继承）属性
