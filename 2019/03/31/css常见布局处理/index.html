css常见布局处理

发表于 2019-03-31 | 更新于 2019-05-09 | 分类于 css | 阅读次数：

总结一波常见的布局吧，作为一名前端工程师，html和css要非常熟练才行a~

布局

水平居中

  • flex+justify-content:center
  • 行内元素使用 text-align:center
  • 块级元素可使用 margin: 0 auto

垂直居中

  • flex+align-items:center
  • 单行文本 line-height+height
  • table-cell+vertical-align:middle
  • absolute+tranform

水平垂直居中，建议使用transform

  • flex+justify-content:center+align-items:center
  • absolute+transform

一列定宽一列自适应

三栏布局–两边定宽中间自适应

不定宽+自适应

等分布局

  • 等百分比宽度
  • flex+justify-content
  • flex+flex-grow:1

等高布局

  • flex + flex:1(默认项：align-items: stretch)

  • table – 每列等宽，每行等高
